# 청년적금 갈아타기 손익계산기 — 설계 문서

작성일: 2026-06-19
연구 데이터: [docs/research/bank-rate-data.md](../../research/bank-rate-data.md)

## 1. 목적 / 문제

청년도약계좌(만기 5년)에서 청년미래적금(만기 3년)으로 **갈아탈지** 고민하는 청년이, 이자 계산기를 따로 펼치거나 상품별로 일일이 계산하지 않고도 **"지금 갈아타면 이득인지 손해인지, 금액으로 얼마인지"** 를 한 화면에서 확인하고 합리적으로 결정하게 한다. 신규 가입자를 위한 두 상품 비교도 함께 제공한다.

## 2. 핵심 도메인 규칙 (정확성의 근거)

### 2.1 갈아타기 = 연계 특별중도해지 (페널티 0)
청년미래적금 상품안내 원문 근거. 도약계좌 보유자가 **연계가입 목적으로** 도약계좌를 해지하면 일반중도해지가 아니라 **특별중도해지**로 처리된다:
- 이자: **기본금리 + 우대금리** 지급 (이미/일부 충족한 도약 우대요건도 인정)
- **정부기여금 + 그 이자 전액 지급**
- **이자소득 비과세 유지**

→ 갈아타기 모드에서 도약계좌 해지환급금은 **그동안 쌓은 가치를 손실 없이 회수**. "중도해지금리·기여금 박탈·과세" 가정은 갈아타기 모드에서 사용하지 않는다. (참고용 '단순(일반)해지' 비교는 범위 외 — §9.)

### 2.2 적금 이자 (자유적립식·단리, 매월 정액 가정) — ★ 공식 예제로 검증됨(§11)
- 만기 원금 = 월납입 P × 개월수 n
- 만기 이자 = P × (연이율 r / 12) × n(n+1)/2  ← 적립식 단리(1회차 n개월…n회차 1개월 예치, 합계계수 n(n+1)/2)
- **원금에 붙는 r = 적용금리 = 기본금리 + 체크된 우대금리 합** (§2.4)
- **정부기여금에 붙는 이자는 기본금리(우대금리 제외)로 계산** — 공식 워크드 예제와 정확히 일치(§11). 기존에 기여금에도 우대금리를 적용하던 가정은 폐기.
- 두 상품 모두 만기/특별해지 시 **비과세** → 만기 이자에 세금 0. (일반중도해지 과세율 15.4% — 갈아타기엔 미적용)

### 2.3 정부기여금
- **청년미래적금**: 월 기여금 = min(P × 매칭률, 월한도). 일반형 6%/월 3만, 우대형 12%/월 6만, 미지급형 0. 합 = 월기여금 × n. 기여금도 같은 r로 이자 발생(적금과 동일 단리식).
- **청년도약계좌**: 소득구간별 매칭표(월 최대 33,000 / 29,000 / 25,200 / 21,000 / 0원). 한도내·추가구간 2단 매칭 — [research](../../research/bank-rate-data.md) 표 그대로 구현.

**소득 입력 정책(확정):** 연봉 숫자를 받지 않는다. 사용자가 **구간·유형을 직접 선택**한다 — 프라이버시·단순성 우선.
- 도약 기여금: **소득구간 드롭다운**(총급여 2,400 / 3,600 / 4,800 / 6,000만 이하, 7,500만 이하=미지급)
- 미래 기여금: **유형 드롭다운**(일반형 6% / 우대형 12% / 미지급형) — 우대형은 중소기업 재직·신규취업 안내 문구 동반
- 소득+우대금리(0.5%p): **체크박스**(총급여 3,600만 이하 충족 시) — §2.4 우대금리 체크에 포함
- (참고) 가구 중위소득 등 가입 "자격" 판정은 수령액 계산과 무관하므로 범위 밖(§9).

### 2.4 은행 + 우대금리
- 사용자가 **은행 선택** → 기본금리 자동 적용.
- 우대금리는 **체크박스(칩)** 로 제공, 충족한 항목의 %p가 합산되어 적용금리에 반영.
- 미래적금의 **연계가입 특별우대(최대 +1.0%p, 은행별 상이)** 는 갈아타기 모드에서 기본 체크 상태로 노출(갈아타기 = 연계가입이므로).
- 데이터는 [research/bank-rate-data.md](../../research/bank-rate-data.md)를 `data/banks.ts`로 구조화. 하단에 "2026.6 공시 기준" 명시.

## 3. 비교 모델 (갈아타기 모드)

분기 시점: 도약 m개월 납입 후 현재.

| | 유지(KEEP) | 갈아타기(SWITCH) |
|---|---|---|
| 기존 잔액 V₀ | 도약에 계속, 만기(60개월)까지 4.5%+우대 | 특별해지로 회수(손실 0), 이후 재예치율 i(기본 0%) |
| 향후 월납입 | 도약 계속(≤70만, 4.5%+우대, 도약 기여금) | 미래적금(≤50만, 5%+우대+연계, 미래 기여금) |
| 만기 | 60개월 | 36개월 |

**비교 시점(horizon) = 미래적금 만기 = 지금 + 36개월.** 이 시점에서 양쪽 **총자산(= 계좌가치 + 보유현금)** 을 비교.

★ **공정성 원칙 — 매달 같은 금액을 저축한다고 전제(동일 out-of-pocket).** 두 상품의 월 한도가 다르므로(도약 70만 / 미래 50만), 월 저축액을 통일하지 않으면 "내 돈을 더 많이 넣은 쪽"이 무조건 자산이 커 보이는 착시가 생긴다. 기준 월저축액 = 현재 도약 월납입액으로 둔다.
- **KEEP 총자산** = 36개월 시점까지 도약에 쌓인 원금+이자(기본+우대)+기여금+기여금이자(기본금리). 36개월 시점에 도약이 아직 만기 전이면 경과분 평가(개월수 = min(기납입m + 36, 60)).
- **SWITCH 총자산** = ① 회수한 V₀(도약 특별해지환급금) ×(1+i) + ② 미래적금 36개월 만기수령액 + ③ **남긴 현금**: 매달 (도약월납입 − 미래월납입)만큼 미납입분이 현금으로 남음 → 36개월 적립(재예치율 i, 기본 0%).
- **갈아타기 손익 = SWITCH − KEEP** (양수면 이득).
- 각 상품의 **실제 만기일·만기 수령액**도 표에 별도 표기(도약 60개월 만기 포함).
- 회수금·남긴현금 재예치율 i 기본 0%(보수적). 사용자가 예금금리 입력 가능.
- (MVP 단순화: KEEP에서 도약이 36개월 시점 이전에 만기되는 경우(기납입 m>24)의 만기 후 현금 재예치는 생략 — 보수적. §9.)

## 4. UX / 화면 구조 (B+C 결합형)

상단부터:
1. **모드 탭**: 🔄 갈아타기 손익 / 🆕 신규 가입 비교
2. **결론 배너 (C)**: "갈아타면 ＋OOO만원 이득 ▲ (3년 뒤 기준)" — 부제에 핵심 가정 요약
3. **비교 타임라인 (②안 — 두 줄 비교)**: "유지하면 → 도약 5년 만기" / "갈아타면 → 미래적금 3년 만기" 두 막대 + 점선 한 개로 "↑ 3년 뒤 비교" 표시. 입력에 따라 막대·마커 실시간 갱신.
4. **입력/결과 (B, 좌우 2열, 실시간)**:
   - 좌: ① 현재 보유 도약계좌(은행·우대금리 체크·월납입·기납입 개월·**소득구간 드롭다운**), ② 갈아탈 미래적금(은행·우대금리 체크칩(연계우대 포함)·월납입·**기여금 유형 드롭다운**). 연봉 숫자 입력 없음. 갈아타기 모드엔 해지사유/중도해지금리 입력 **없음**.
   - **월 납입액은 슬라이더(range)** 로 입력하고 현재 금액을 실시간 표시 → 슬라이더 조정 시 결론 배너·차트·분해표가 즉시 재계산(React 반응형).
   - 우: 막대 비교 + 손익 분해표(도약 회수액, 미래 적금이자, 정부기여금 등) + "▾ 월별 계산 근거·가정 펼치기"
5. 하단: 금리 출처·면책 문구.

신규 비교 모드: 보유 입력 없이 두 상품을 동일 조건(월납입·소득구간)으로 만기 수령액 비교.

## 5. 아키텍처 (React + TypeScript + Vite, 정적 배포)

```

  domain/savings.ts    순수 계산: 적금이자·기여금·만기수령액·특별해지환급금
```

## 6. 데이터 / 상태
- 은행·금리 데이터는 빌드 타임 상수(`data/`).
- 입력값을 URL 쿼리에 직렬화 → 서버 없이 결과 링크 공유.
- 금리·기여금은 2026.6 공시 기준 내장값. "실제 적용금리는 가입 은행 확인" 면책 문구.

## 7. 에러 / 엣지 케이스
- 월납입 한도 초과(도약 70만 / 미래 50만), 기납입 개월 > 만기, 음수 입력 → 인라인 경고.
- 기여금 미지급 구간(고소득)·외국인(기여금 미지급, 비과세만) 토글 반영.
- 우대금리 합이 은행별 최고 우대 상한을 넘지 않도록 클램프.

## 8. 테스트
- Vitest 단위 테스트로 `domain/savings.ts`·`compare.ts` 검증. **§11의 공식 골든 케이스를 회귀 테스트로 고정**(허용오차: 만원 미만 반올림).
  - 적금 단리 이자 공식(14.4%→세전이자 400만, 19.4%→538만)
  - 이자소득세 15.4%(→62만 / 83만)
  - 미래 기여금(일반 108만 / 우대 216만), 도약 기여금(33,000원/월·198만)
  - 미래적금 만기 총수령(일반 2,138만 / 우대 2,255만, 8% 가정) — **원금 8% + 기여금 5%(기본)**
  - 특별해지 환급금 = 원금+비과세이자+기여금
  - KEEP vs SWITCH 36개월 손익 부호/금액
- 컴포넌트는 입력→결과 흐름 스모크 테스트 1~2개.

## 11. 공식 자료 검증 (신뢰성 근거) — 2026.06.11 정책브리핑 / 금융위 Q&A 대조

산식을 공식 워크드 예제와 산수로 대조함(스크립트 검증 완료). **모두 원 단위까지 일치.**

| 검증 항목 | 산식 결과 | 공식 값 | 일치 |
|---|---|---|---|
| 50만×36, 14.4% 단리 세전이자 | 3,996,000 | 400만 | ✅ |
| 이자소득세 15.4% | 615,384 | 62만 | ✅ |
| 19.4% 세전이자 / 세금 | 5,383,500 / 829,059 | 538만 / 83만 | ✅ |
| 미래 일반형 기여금(6%,월3만) | 1,080,000 | 108만 | ✅ |
| 미래 우대형 기여금(12%,월6만) | 2,160,000 | 216만 | ✅ |
| 미래 일반형 만기총액(원금8%+기여금5%) | 21,383,250 | 2,138만 | ✅ |
| 미래 우대형 만기총액 | 22,546,500 | 2,255만 | ✅ |
| 도약 기여금(소득2,300만·월70만): 40만×6%+30만×3% | 24,000+9,000=33,000/월 | 33,000 | ✅ |
| 도약 기여금 60개월 | 1,980,000 | 198만 | ✅ |

**검증으로 확정된 산식 규칙:**
1. 적립식 단리 이자 = P×(r/12)×n(n+1)/2.
2. 원금은 적용금리(기본+우대), **정부기여금은 기본금리**로 이자 계산. (기여금에 우대금리 적용 시 우대형이 +10만 과대 → 폐기)
3. 갈아타기(연계 특별중도해지)는 기여금·우대금리·비과세 **전부 유지**(§2.1) — 공식 Q8·정책브리핑 재확인.
4. 미래적금은 **가입 후 소득 재심사 없음**(전년도 소득 기준 1회 심사) → 기여금 유형 가입 시 고정. (도약계좌는 매년 유지심사 — 다름.) 단, 미래 우대형은 만기 1달 전까지 중소기업 29개월 근속요건.

**⚠️ 출처 간 날짜 불일치(미해결 — 단정 금지):** 연구노트(서민금융진흥원 HTML)는 최초가입 '26.7.27~8.7로, 정책브리핑(2026.06.11)은 **출시 6.22 · 신청 6.22~7.3 · 갈아타기 6월만 허용**으로 기재. 정책브리핑이 더 최신/권위. → 앱은 특정 날짜를 사실로 박지 말고 **출처·기준일을 표기**하고, 갈아타기 안내는 "최초 가입기간 한정"으로 문구화. (오늘 기준 갈아타기 창은 임박/진행 중 — 안내 배너에 반영.)

## 9. 범위 밖 (YAGNI)
- 로그인 / 서버 / DB / 실시간 금리 API
- 도약계좌 일시납입·부분인출 시뮬(복잡도 큼 — 후순위 확장)
- 일반(단순)중도해지 손해 계산(갈아타기엔 불필요, 참고기능으로 추후)
- 3개 이상 상품 비교, 다년 변동금리 정밀 모델
- **부분충족 우대금리 인정(후순위):** 원문 각주상 "특별중도해지 시점엔 일부만 충족했어도 만기까지 가면 충족되는" 도약 우대요건도 인정됨. MVP에선 사용자가 현재 충족한 우대만 체크. "충족 예정" 처리는 추후 추가.
